MySQL en cPanel

SQL es el estándar internacional para realizar peticiones y escritura de información en bases de datos. MySQL es esencialmente un servidor SQL ya que responde a las peticiones de información SQL. Puedes comunicarte con MySQL usando una amplia variedad de programas y lenguaje (PHP es el más común en sistios web).

MySQL es de libre distribución y por tanto es de uso libre.

Para crear una base de datos siga los siguientes pasos:

  1. Seleccione el menú Bases de datos MySQL que aparece en el menú principal del cPanel.
  2. Escriba el nombre de la base de datos que desea realizar en el campo correspondiente.
  3. Posteriormente oprima una vez el botón Crear base de datos MySQL.

Nota: tendrá que crear al menos un usuario para la base de datos y otorgarle todos los privilegios (ALL check oprimido) para poder usar la nueva base de datos.

¿Como crear usuario MySQL?

Para crear una usuario de base de datos MySQL siga los siguientes pasos:

  1. Seleccione el menú Bases de datos MySQL que aparece en el menú principal del cPanel.
  2. Escriba el nombre de usuario y contraseña apropiada.
  3. Oprima en Crear usuario MySQL.
¿Como borrar una base de datos MySQL?

Puede borrar una base de datos si ya no la va a usar. Debe estar seguro de que se ha guardado la información de la base de datos a no ser que la vayaa a desechar. Si se borra una base de datos no se borrarán los usuarios pero sí sus privilegios.

Para borrar una base de datos MySQL siga los siguientes pasos:

  1. Seleccione el menú Bases de datos MySQL que aparece en el menú principal del cPanel.
  2. Selecciona el botón Delete correspondiente a la base de datos a borrar.
¿Como otorgar privilegios a un usuario?

Para otorgar privilegios a un usuario en una base de datos MySQL siga los siguientes pasos:

  1. Seleccione el menú Bases de datos MySQL que aparece en el menú principal del cPanel.
  2. Escoja a un usuario concreto de la lista y también una base de datos correcta.
  3. Oprima la casilla “All” en caso de agregar todos los privilegios, si no solo seleccione los privilegios necesarios.
  4. Finalmente pulsa una vez “Otorgar permisos”.
¿Como respaldar una base de datos MySQL?

Para respaldar o exportar una Base de Datos desde phpMyAdmin debe seguir los siguientes pasos:

  1. Ingrese a cPanel de su sitio.
  2. Busque el ícono de phpMyAdmin en la sección Bases de Datos o Databases.
  3. Seleccione la Base de Datos a Exportar.
  4. Pulse la barra Exportar del recuadro principal.
  5. Seleccione la opción Todas (All) o bien pulse sobre las tablas que desee exportar.
  6. Marque como habilitada la casilla Save as file o (Guardar Como) e ingrese un nombre.
  7. Pulse el botón Ir (Go) y la Base de Datos está respaldada ahora.
¿Cómo recuperar contraseña de usuario MySQL?

Si ha olvidado su contraseña de Base de Datos, lamentablemente no es posible recuperarlo, pero NO todo está perdido. Puedes realizar los siguientes pasos:

  1. Ingrese a cPanel de su sitio.
  2. Busque el ícono de phpMyAdmin en la sección Bases de Datos o Databases.
  3. Cree un Nuevo Usario con la contraseña que desee.
  4. Añada dicho usuario a la Base de Datos deseada.
  5. Reemplace ahora en sus archivos de confuración el usuario antiguo por el usuario nuevo.

Esto debería resolver el problema de conectividad. De no ser así debes consultar a tu programador o en su caso con el área de soporte de Daitum.

¿Cómo saber la versión de MySQL?

Si desea conocer qué versión de MySQL se ejecuta en su sitio, sólo debes ingresar en su cPanel y en la columna izquiera podrá encontrarlo. En algunos casos debe pulsar Expand Stats o Expandir Estadísticas.

¿Cómo hacer un String (cadena) de conexión?

Para conectar su base de datos puede usar el siguiente string o cadena o bien, los datos contenidos aquí:

Server=suServidor;Database=suDataBase;Uid=suUsuario;Pwd=suContraseña;Port=3306

Donde:

  • suServer debe ser sustituido por la IP del servidor, el nombre del dominio o localhost, dependiendo el ambiente.
  • suDatabase debe ser sustituido por el prefijo del nombre de usuario del panel de control seguido del nombre de la base datos, del tipo (userCpanel_NombreBasedeDatos).
  • suUsuario debe ser ustituido por el prefijo del nombre de usuario del panel de control seguido del nombre de usuaro, del tipo (userCpanel_UsuarioBasedeDatos).
  • suContraseña debe ser sustituido por la contraseña del usuario de la base de datos.
¿Cómo optimizar o reparar una base de datos?

Es una buena práctica optimizar las bases de datos cuando éstas corren por debajo de su desempeño habitual. En algunos otros casos, falta de memoria o múltiples procesos concurrentes podrían dañarlas y adicionalmente sería requerido repararlas. En estos casos recomendamos el siguiente proceso:

ES INDISPENSABLE HACER UN BACKUP DE LA BASE DE DATOS ANTES

  • Ingresar a su panel de control (cPanel).
  • En la Sección Bases de Datos (o Databases) ubique el ícono phpMyAdmin e ingrese en él.
  • En la columna izquierda se encuentran la lista de Bases de Datos con las que cuenta su sitio web y debe seleccionar aquellas que desee Reparar.
  • En la parte superior de la lista de Bases de Datos está la opción Check All o Seleccionar Todas y a la derecha de dicho botón hay un menú desplegable donde debe seleccionar Reparar o Repair.
  • Una vez hecho esto y cuando se haya terminado el proceso volverá al paso previo y puedes repetirlo ahora seleccionando Optimizar (Optimize).
