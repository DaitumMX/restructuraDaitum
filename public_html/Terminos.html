Términos de Servico

Antes de iniciar cualquier proyecto es necesario que lea cuidadosamente los siguientes Términos:

Los pagos de cada proyecto se realizan en dos exhibiciones, el 50% antes de iniciar el proyecto y el 50% restante al finalizarlo. La factura se enviará únicamente después de que el cliente haya realizado el pago total del proyecto.

En caso de que el cliente ya cuente con hosting y dominio es necesario que envíe a Daitum toda la información que se requiera para realizar el desarrollo del sitio.

Daitum empezará a trabajar en su proyecto únicamente después de que se haya recibido el depósito del anticipo y toda la información necesaria para el desarrollo de la pagina web.

Los tiempos estimados de entrega empiezan a contar únicamente después de que Daitum haya recibido el pago inicial, toda la información que irá en la pagina web y que el cliente haya aceptado alguna de las propuestas de diseño o template según sea el caso.

Daitum no se hace responsable de imágenes, textos o cualquier otro tipo de contenido proporcionados por el cliente. Cada servicio adicional que se requiera como edición de imágenes, revisión de ortografía en textos, vectorización de logotipos, se cotizarán aparte.

Dependiendo del tipo de proyecto Daitum se reserva el derecho de usar un template como base para el desarrollo de la pagina web.

Una vez que el cliente haya aceptado el diseño de la pagina web y Daitum haya iniciado con el desarrollo, cualquier cambio extra será cotizado por separado.

En caso de contratar el hosting y/o dominio con Daitum al finalizar el año se deberá realizar la renovación anual del servicio contratado aplicando las características y tarifas vigentes.

Daitum se reserva el derecho de cancelar el proyecto si después de 30 días naturales de haber recibido el pago inicial, no haya recibido la información completa por parte del cliente, o no pueda terminar el proyecto durante este tiempo por cuestiones ajenas a Daitum o en su caso aplicará una penalización del 25% del valor del paquete o servicio contratado. En caso de cancelación, Daitum es dueño de cualquier trabajo realizado hasta la fecha de cancelación.

Una vez transcurrido un año se deberá realizar la renovación anual del servicio de alojamiento web (web hosting) de acuerdo a las características y tarifa del plan correspondiente, así como los derechos del nombre de dominio que corresponda.

Ningún anticipo es reembolsable ni transferible.

Una vez finalizado el proyecto, Daitum entregará todos los datos de acceso y documentación al cliente únicamente después de haber recibido el pago final. Después de notificar al cliente que el proyecto fue finalizado, si en un plazo de 3 días hábiles el cliente no a realizado el último pago, Daitum dará de baja el sitio hasta que el cliente no efectúe el pago completo por el servicio contratado.

Al contratar a Daitum para su proyecto, esta aceptando las políticas de servicio.
